# Tic Tac Toe Game

## Description
This project is a Tic Tac Toe game developed as part of a web development task. It is a simple interactive game where two players can play and take turns to mark X and O on the grid.

## Objective
The objective of this task is to understand basic game logic, user interaction, and DOM manipulation using HTML, CSS, and JavaScript.

## Technologies Used
- HTML
- CSS
- JavaScript
